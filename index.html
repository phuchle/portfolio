Phuc's Pomodoro

The Pomodoro technique is a time management method, where work is broken down into 25 minute sessions separated by 5 minute breaks. This app is a timer which allows the user to customize their work/break sessions and plays a chime at the beginning and end of each session. The timer is a button which plays/pauses the session.

Which Twitch?

Users can view the status of Twitch streamers with this app and filter between online and offline users. The Twitch API recently changed, and I had to use a proxy API in order to create this app. I continued to use flexbox to align this page and CSS transitions along with jQuery in order to create the filtering functionality.

Lowdown on Local Weather

I used the Weather Underground API and the user location to display local temperature and weather conditions within the city. I used minimal jQuery in order to gain practice with pure JS selectors.

While building the functions to show the weather, I learned more about how asynchronous JavaScript loads, since at one point my code was returning undefined for the temperature. The rest of the JavaScript had already loaded before my API call returned a value, so I had to restructure my code to run only after the asynchronous portion of the code was finished.
