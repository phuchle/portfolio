Recipeas

Recipeas (pun intended) allows users to create, store, edit, and delete recipes without needing to connect to a server. The application stores the data as a string in the browser's local storage. Recipeas was created with Create-React-App and state is handled with Redux. Upcoming features include a redesigned recipe creation flow and nutritional information for recipes.

Leaderboard

This app shows top scoring users from freecodecamp.com, an open source online web development course. The users can be sorted by their all time points or their recent points (past 30 days). The API call is made with axios and then parsed into rows. The sort toggling is handled by the table headers, which call functions to render the corresponding data.

Unbeatable Tic-Tac-Toe

This is the classic game of tictactoe, played against an unbeatable AI--it will either win or draw the game. The AI uses a minimax function to recursively analyze all possible moves and end-game scenarios to determine the best move. It also continues to make optimal plays even when it cannot win.

Simon

Simon is a game where a pattern plays and the player has to enter the same pattern. One additional color/sound combination is added to this pattern. If a player can get to 20, they will win! Strict mode forces the player to start over at 1 if they make a mistake entering the pattern.
