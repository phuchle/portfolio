Wacky Wiki Window

This app allows the user to either view a random Wikipedia page or search wikipedia for a search term and have it displayed on the same page. I employed flexbox in order to align the contents of page. This approach was much more simple and a intuitive for me when aligning items in CSS.

Which Twitch?

Users can view the status of Twitch streamers with this app and filter between online and offline users. The Twitch API recently changed, and I had to use a proxy API in order to create this app. I continued to use flexbox to align this page and CSS transitions along with jQuery in order to create the filtering functionality.

Lowdown on Local Weather

I used the Weather Underground API and the user location to display local temperature and weather conditions within the city. I used minimal jQuery in order to gain practice with pure JS selectors.

While building the functions to show the weather, I learned more about how asynchronous JavaScript load when my code was returning undefined for the temperature. The rest of the JavaScript had already loaded before my API call returned a value, so I had to restructure my code to run only after the asynchronous portion of the code was finished.

Random Quote Machine

This app retrieves a random quote from the QuotesDesign API and displays it for the user. This quote can be Tweeted out to the user's timeline. I used HTML/CSS and jQuery to dynamically retrieve the quote information and create the tweet-able text.
